About This System

This is a QR/barcode-based student sign-in system for Africa Nazarene University. Records are stored securely in Firebase Firestore — persistent across browsers and devices.

  • Designed for library, lab, and event attendance tracking.
  • Cloud storage via Firebase — no data lost when browser clears.
  • Secure admin login with email and password via Firebase Auth.
  • Export records to CSV from the Admin dashboard any time.
